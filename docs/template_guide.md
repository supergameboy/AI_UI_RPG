# 游戏模板编辑器使用指南

## 概述

游戏模板编辑器是 AI-RPG Engine 的核心功能，用于创建和编辑游戏模板。模板定义了游戏的世界观、角色创建规则、游戏规则、AI 行为约束等所有内容。通过模板编辑器，您可以：

- 创建独特的游戏世界观和背景设定
- 定义种族、职业、背景等角色创建选项
- 配置战斗、技能、物品、任务等游戏规则
- 设置 AI 的行为约束和响应风格
- 设计起始场景和初始内容
- 自定义 UI 主题和界面布局

编辑器采用左侧导航栏 + 右侧内容区的布局结构，点击左侧导航项即可切换到对应的编辑模块。

---

## 编辑器模块说明

### 1. 基础信息编辑

基础信息是模板的核心元数据，用于标识和描述模板。

**可编辑字段：**

| 字段 | 说明 | 示例 |
|------|------|------|
| 模板名称 | 模板的显示名称 | "奇幻冒险" |
| 描述 | 模板的详细描述 | "一个充满魔法与冒险的奇幻世界" |
| 版本号 | 模板版本 | "1.0.0" |
| 作者 | 模板作者名称 | "AI-RPG Team" |
| 游戏模式 | 游戏的核心玩法模式 | 见下表 |
| 标签 | 用于分类和搜索的标签 | ["奇幻", "魔法", "冒险"] |

**游戏模式选项：**

| 模式 | 说明 |
|------|------|
| 文字冒险 | 传统的文字冒险游戏，注重叙事和选择 |
| 回合制RPG | 回合制战斗系统，注重策略和数值 |
| 视觉小说 | 以剧情为主的视觉小说风格 |
| 动态战斗 | 实时或半实时战斗系统 |

**操作说明：**
- 输入标签后按回车键或点击「添加」按钮添加标签
- 点击标签右侧的关闭按钮可删除标签

---

### 2. 世界观设定

世界观设定定义了游戏世界的背景和特色，是 AI 生成内容的重要依据。

**可编辑字段：**

| 字段 | 说明 | 是否必填 |
|------|------|----------|
| 世界名称 | 游戏世界的名称 | 是 |
| 世界描述 | 世界的背景、特色和氛围描述 | 是 |
| 时代背景 | 世界所处的时代 | 否 |
| 科技水平 | 世界的科技发展程度 | 否 |
| 魔法系统 | 魔法的类型、规则和限制 | 否 |
| 自定义字段 | 额外的世界观设定 | 否 |

**自定义字段：**
- 点击「添加」按钮创建新的自定义字段
- 可以编辑字段名称和值
- 点击删除按钮移除字段

**AI 生成功能：**
- 点击「AI 生成世界观」按钮可自动生成世界观设定
- 可以输入描述提示词，AI 将根据描述生成
- 留空则 AI 会自动生成完整的世界观
- 生成结果可预览，选择「应用」或「重新生成」

---

### 3. 种族编辑

种族定义了玩家角色可以选择的种族类型，每个种族有独特的属性加成和能力。

**种族属性：**

| 属性 | 说明 |
|------|------|
| 种族名称 | 种族的显示名称 |
| 描述 | 种族的背景和特色描述 |
| 属性加成 | 该种族在特定属性上的加成值 |
| 属性惩罚 | 该种族在特定属性上的惩罚值 |
| 特殊能力 | 种族特有的能力或特性 |
| 可选职业 | 该种族可以选择的职业列表 |

**操作说明：**
1. 点击「添加种族」按钮创建新种族
2. 填写种族名称和描述
3. 设置属性加成和惩罚（数值范围：0-10）
4. 添加特殊能力（输入后按回车或点击添加）
5. 勾选该种族可选的职业
6. 点击「保存」完成编辑

**AI 生成功能：**
- 点击「AI 生成」按钮可自动生成种族
- AI 会根据已设置的世界观自动生成符合背景的种族
- 可以输入描述提示词定制生成内容
- 生成结果可预览后选择「添加到列表」或「重新生成」

---

### 4. 职业编辑

职业定义了玩家角色的职业类型，决定了角色的核心能力和成长方向。

**职业属性：**

| 属性 | 说明 |
|------|------|
| 职业名称 | 职业的显示名称 |
| 描述 | 职业的特色和定位描述 |
| 生命骰 | 决定生命值成长的骰子类型 |
| 主属性 | 职业的核心属性（可多选） |
| 技能熟练 | 职业擅长的技能列表 |
| 初始装备 | 职业开始时的装备 |

**生命骰选项：**
- d4：脆弱型职业（如法师）
- d6：轻型职业（如盗贼）
- d8：中型职业（如游侠）
- d10：重型职业（如战士）
- d12：坦克型职业（如野蛮人）

**操作说明：**
1. 点击「添加职业」按钮创建新职业
2. 填写职业名称和描述
3. 选择生命骰类型
4. 点击属性按钮选择主属性（可多选）
5. 点击「保存」完成编辑

**AI 生成功能：**
- 点击「AI 生成」按钮可自动生成职业
- AI 会根据世界观和种族设定生成匹配的职业
- 支持描述提示词定制

---

### 5. 背景编辑

背景定义了角色的出身和经历，提供额外的技能和特性。

**背景属性：**

| 属性 | 说明 |
|------|------|
| 背景名称 | 背景的显示名称 |
| 描述 | 背景的故事和特色 |
| 技能熟练 | 背景提供的技能熟练项 |
| 语言 | 背景提供的语言能力 |
| 初始装备 | 背景提供的初始物品 |
| 背景特性 | 背景的特殊能力或特点 |

**操作说明：**
1. 点击「添加背景」按钮创建新背景
2. 填写背景名称和描述
3. 添加技能熟练项和语言
4. 填写背景特性
5. 点击「保存」完成编辑

**AI 生成功能：**
- 点击「AI 生成」按钮可自动生成背景
- AI 会根据世界观生成符合设定的背景选项
- 支持描述提示词定制

---

### 6. 属性编辑

属性系统定义了角色的核心属性，是种族加成和职业主属性的基础。

**默认属性：**

| 属性 ID | 名称 | 缩写 | 说明 |
|---------|------|------|------|
| strength | 力量 | STR | 物理力量和近战能力 |
| dexterity | 敏捷 | DEX | 灵活性和远程能力 |
| constitution | 体质 | CON | 生命力和耐力 |
| intelligence | 智力 | INT | 学习能力和魔法天赋 |
| wisdom | 感知 | WIS | 直觉和洞察力 |
| charisma | 魅力 | CHA | 社交能力和影响力 |

**属性配置：**

| 字段 | 说明 |
|------|------|
| 属性 ID | 属性的唯一标识符（用于种族加成和职业主属性） |
| 属性名称 | 属性的显示名称 |
| 属性缩写 | 属性的简短缩写（显示在界面上） |
| 描述 | 属性的详细说明 |
| 最小值 | 属性的最低可能值 |
| 默认值 | 属性的初始值 |
| 最大值 | 属性的最高可能值 |

**操作说明：**
1. 点击「添加属性」按钮创建新属性
2. 填写属性 ID（创建后不可修改）、名称和缩写
3. 设置数值范围和默认值
4. 点击「保存」完成编辑

**注意事项：**
- 属性 ID 应与种族加成和职业主属性保持一致
- 删除属性会影响已配置的种族和职业设定
- 建议在配置种族和职业前先完成属性系统设置

---

### 7. 规则配置

规则配置定义了游戏的核心规则系统，包括战斗、技能、物品、任务等方面。

#### 7.1 战斗规则

| 规则 | 说明 | 选项 |
|------|------|------|
| 战斗类型 | 战斗的核心机制 | 叙事型、回合制、实时、混合 |
| 先攻类型 | 决定行动顺序的方式 | 敏捷、随机、自定义 |
| 行动点数 | 每回合可执行的行动次数 | 1-10 |
| 暴击阈值 | 触发暴击的最低数值 | 1-20 |
| 暴击倍率 | 暴击时的伤害倍数 | 1-10 |

#### 7.2 技能规则

| 规则 | 说明 |
|------|------|
| 最大等级 | 技能可达到的最高等级 |
| 冷却系统 | 技能冷却的计算方式（回合制/时间制/无冷却） |
| 升级基础消耗 | 技能升级的基础消耗值 |
| 升级消耗倍率 | 每级消耗的增长倍率 |

#### 7.3 物品规则

| 规则 | 说明 |
|------|------|
| 最大槽位 | 背包的最大容量 |
| 重量系统 | 是否启用物品重量限制 |

#### 7.4 任务规则

| 规则 | 说明 |
|------|------|
| 最大同时任务数 | 玩家可同时进行任务的上限 |
| 时间系统 | 是否启用任务时间限制 |

#### 7.5 数值复杂度

| 级别 | 说明 |
|------|------|
| 简单 | 简化的数值系统，适合新手玩家 |
| 中等 | 平衡的数值复杂度，兼顾深度和易用性 |
| 复杂 | 详细的数值系统，适合硬核玩家 |

#### 7.6 特殊规则

| 规则 | 说明 |
|------|------|
| KP（守密人）系统 | 启用守密人系统，由 AI 扮演 KP 角色 |
| 永久死亡 | 角色死亡后无法复活，增加游戏紧张感 |
| 存档限制 | 存档的限制方式（无限制/仅检查点/仅手动/铁人模式） |
| 自定义规则 | 游戏特定的自定义规则列表 |

### 8. AI 约束设置

AI 约束设置控制 AI 生成内容的行为和风格，确保游戏体验符合预期。

#### 8.1 基调设置

| 基调 | 说明 |
|------|------|
| 严肃 | 正式、庄重的叙事风格 |
| 幽默 | 轻松、有趣的叙事风格 |
| 黑暗 | 阴暗、沉重的叙事风格 |
| 浪漫 | 浪漫、感性的叙事风格 |
| 自定义 | 用户自定义的叙事风格 |

#### 8.2 内容分级

| 分级 | 说明 |
|------|------|
| 全年龄 | 适合所有年龄段玩家 |
| 青少年 | 适合青少年及以上玩家 |
| 成人 | 仅适合成年玩家 |

#### 8.3 AI 行为配置

**响应风格：**

| 风格 | 说明 |
|------|------|
| 叙事风格 | AI 以故事叙述的方式回应，注重情节和氛围 |
| 机械风格 | AI 以简洁、直接的方式回应，注重游戏机制 |
| 自适应 | AI 根据情境自动调整回应风格 |

**详细程度：**

| 级别 | 说明 |
|------|------|
| 简略 | 简洁的描述，快速推进剧情 |
| 正常 | 适中的详细程度，平衡叙事和节奏 |
| 详细 | 丰富的细节描写，沉浸式体验 |

**玩家自由度：**

| 级别 | 说明 |
|------|------|
| 引导式 | AI 提供明确的选项和建议，引导玩家决策 |
| 平衡 | AI 提供适度的引导，同时保持玩家自由度 |
| 自由形式 | AI 给予玩家最大的自由，不提供明确引导 |

#### 8.4 内容约束

| 设置 | 说明 |
|------|------|
| 禁止主题 | AI 将不会生成与这些话题相关的内容 |
| 必需元素 | AI 将确保故事中包含这些元素 |

**操作说明：**
- 输入话题后按回车或点击「添加」按钮添加
- 点击话题右侧的关闭按钮可删除

### 9. 起始场景

起始场景定义了玩家开始游戏时的初始环境和内容。

#### 9.1 地点信息

| 字段 | 说明 |
|------|------|
| 地点名称 | 起始地点的名称 |
| 场景描述 | 玩家开始时的场景描述 |

#### 9.2 NPC 定义

通过标签页切换到 NPC 编辑：

| 属性 | 说明 |
|------|------|
| 名称 | NPC 的显示名称 |
| 头衔 | NPC 的头衔或职业 |
| 描述 | NPC 的外观和特色描述 |
| 角色 | NPC 的类型（商人/任务发布者/敌人/盟友/中立/自定义） |
| 性格 | NPC 的性格特点 |
| 对话 | NPC 的对话内容 |
| 属性 | NPC 的数值属性（等级、生命值、攻击、防御等） |
| 服务 | NPC 提供的服务（商店/旅店/铁匠/治疗/训练） |

**AI 生成功能：**
- NPC 编辑器支持 AI 生成单个 NPC
- 可输入描述提示词定制生成内容

#### 9.3 物品定义

通过标签页切换到物品编辑：

| 属性 | 说明 |
|------|------|
| 名称 | 物品的显示名称 |
| 描述 | 物品的外观和功能描述 |
| 类型 | 物品类型（武器/护甲/饰品/消耗品/材料/任务物品/杂项） |
| 稀有度 | 物品稀有度（普通/优秀/稀有/史诗/传说/独特） |
| 属性 | 物品提供的属性加成 |
| 效果 | 物品的使用效果 |
| 价值 | 物品的买卖价格 |

**AI 生成功能：**
- 物品编辑器支持 AI 生成单个物品
- 可输入描述提示词定制生成内容

#### 9.4 任务定义

通过标签页切换到任务编辑：

| 属性 | 说明 |
|------|------|
| 名称 | 任务的显示名称 |
| 描述 | 任务的背景和目标描述 |
| 类型 | 任务类型（主线/支线/日常/隐藏） |
| 目标 | 任务的完成目标列表 |
| 奖励 | 完成任务的奖励列表 |
| 发布者 | 发布任务的 NPC |
| 时间限制 | 任务的完成时限 |

**AI 生成功能：**
- 任务编辑器支持 AI 生成单个任务
- 可输入描述提示词定制生成内容

#### 9.5 场景 AI 生成

点击「AI 生成场景」按钮可一次性生成完整的起始场景：

- 自动生成地点名称和场景描述
- 自动生成 1-3 个 NPC
- 自动生成 1-3 个物品
- 自动生成 1-2 个初始任务

生成结果可预览后选择「应用到场景」或「重新生成」。

```json
  }
}
```

### 12. 预览测试

预览测试功能允许您在保存前验证模板效果。

**预览内容：**

1. **角色创建预览**
   - 查看所有种族及其属性加成、特殊能力
   - 查看所有职业及其生命骰、主属性、技能
   - 查看所有背景及其特性、技能熟练

2. **初始场景预览**
   - 查看起始地点名称和描述
   - 查看所有 NPC 及其属性、角色类型
   - 查看所有物品及其类型、稀有度、属性
   - 查看所有任务及其目标、奖励

**操作说明：**
- 点击编辑器顶部的「预览测试」按钮打开预览窗口
- 通过标签页切换「角色创建预览」和「初始场景预览」
- 点击「关闭」按钮返回编辑器

## AI 生成功能说明

模板编辑器集成了强大的 AI 生成功能，可以大幅提升模板创建效率。

### 支持的生成类型

| 类型 | 说明 |
|------|------|
| 世界观 | 自动生成世界名称、描述、时代背景、魔法系统等 |
| 种族 | 根据世界观生成种族，包含属性加成和特殊能力 |
| 职业 | 根据世界观生成职业，包含生命骰和主属性 |
| 背景 | 根据世界观生成背景，包含技能熟练和特性 |
| NPC | 生成 NPC 定义，包含属性和对话 |
| 物品 | 生成物品定义，包含属性和效果 |
| 任务 | 生成任务定义，包含目标和奖励 |
| 完整场景 | 一次性生成地点、NPC、物品、任务 |

### 使用前提

- 需要先在设置页面配置 LLM API Key
- 支持的 LLM 提供商：DeepSeek、GLM、Kimi 等

### 使用技巧

1. **先设置世界观**：AI 会根据世界观生成相关内容，确保一致性
2. **使用描述提示词**：输入具体描述可以获得更符合预期的结果
3. **多次生成**：如果不满意，可以重新生成直到获得理想结果
4. **手动微调**：AI 生成的内容可以手动编辑和调整

### 生成流程

```
1. 点击「AI 生成」按钮
      ↓
2. 输入描述提示词（可选）
      ↓
3. 点击「生成」按钮
      ↓
4. 预览生成结果
      ↓
5. 选择「应用」或「重新生成」
```

### 常见问题

**Q: 种族的属性加成不生效？**
A: 确保种族加成中使用的属性 ID 与属性编辑器中定义的 ID 一致。

**Q: AI 生成失败？**
A: 检查是否已配置 LLM API Key，确保网络连接正常。

**Q: 预览显示不完整？**
A: 确保所有必填字段都已填写，检查数据格式是否正确。

**Q: 如何复制现有模板？**
A: 在模板管理界面选择模板，点击「复制」按钮创建副本。

### 设计建议

1. **保持一致性**：世界观、种族、职业应该风格统一
2. **平衡性**：种族和职业的属性应该相对平衡
3. **可玩性**：起始场景应该有足够的内容引导玩家
4. **扩展性**：预留自定义字段和规则的空间
