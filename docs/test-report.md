## 发现的问题与修复

### 问题 1: 开发者面板不显示
- **原因**: `isDeveloperPanelVisible` 默认为 false，开启开发者模式时未自动显示
- **修复**: 在 GlobalDeveloperTools 组件中添加 `showDeveloperPanel()` 调用
- **文件**: `packages/frontend/src/components/developer/GlobalDeveloperTools.tsx`

### 问题 2: 版本号显示过时
- **现象**: 前端显示 v0.2.0，实际应为 v0.3.0
- **状态**: 待更新
