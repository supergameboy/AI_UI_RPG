# 角色创建系统 Spec

## Why

当前游戏缺少角色创建流程，玩家无法创建自己的角色开始游戏。需要实现一个完整的角色创建系统，支持AI随机生成种族/职业/背景，增加游戏的可重玩性和趣味性。

## What Changes

- 新增角色创建界面和流程
- 新增AI生成角色选项功能（种族、职业、背景）
- 新增角色确认和预览界面
- 新增设置项：AI随机生成开关、文生图提示词开关
- 修改开始游戏流程：选择模板 → 角色创建 → 进入游戏
- 扩展Character数据结构，支持自定义属性和外观

## Impact

- Affected specs: 游戏主界面、设置系统、存档系统
- Affected code: 
  - `packages/frontend/src/components/` - 新增角色创建组件
  - `packages/frontend/src/stores/` - 新增角色创建状态管理
  - `packages/backend/src/services/` - 新增角色生成服务
  - `packages/shared/src/types/` - 扩展角色类型定义

## ADDED Requirements

### Requirement: 角色创建流程

系统应提供完整的角色创建流程，玩家通过分步选择创建自己的角色。

#### Scenario: 开始新游戏
- **WHEN** 玩家点击"开始新游戏"
- **THEN** 显示故事模板选择界面
- **AND** 玩家选择一个模板后进入角色创建流程

#### Scenario: 输入角色名称
- **WHEN** 进入角色创建流程
- **THEN** 显示角色名称输入界面
- **AND** 玩家输入名称后进入种族选择

#### Scenario: 选择种族
- **WHEN** 玩家完成名称输入
- **THEN** 显示种族选择界面
- **AND** 显示模板预设的种族选项
- **AND** 如果AI随机生成开启，额外显示AI生成的3个种族选项
- **AND** 玩家选择种族后进入职业选择

#### Scenario: 选择职业
- **WHEN** 玩家完成种族选择
- **THEN** 显示职业选择界面
- **AND** 显示模板预设的职业选项
- **AND** 如果AI随机生成开启，额外显示AI生成的3个职业选项
- **AND** 玩家选择职业后进入背景选择

#### Scenario: 选择背景
- **WHEN** 玩家完成职业选择
- **THEN** 显示背景选择界面
- **AND** 显示模板预设的背景选项
- **AND** 如果AI随机生成开启，额外显示AI生成的3个背景选项
- **AND** 玩家选择背景后进入角色确认

#### Scenario: 确认角色
- **WHEN** 玩家完成背景选择
- **THEN** 显示完整角色卡预览
- **AND** 显示角色名称、种族、职业、背景
- **AND** 显示属性值（AI自动分配）
- **AND** 显示外观描述
- **AND** 显示文生图提示词（如果设置开启）
- **AND** 玩家可以确认开始游戏或返回修改

### Requirement: AI随机生成角色选项

系统应支持AI根据模板世界观随机生成种族、职业、背景选项。

#### Scenario: 生成种族选项
- **WHEN** AI随机生成开启且玩家进入种族选择
- **THEN** AI根据模板世界观生成3个种族选项
- **AND** 每个种族包含：名称、描述、属性加成、特殊能力

#### Scenario: 生成职业选项
- **WHEN** AI随机生成开启且玩家进入职业选择
- **THEN** AI根据模板世界观和已选种族生成3个职业选项
- **AND** 每个职业包含：名称、描述、主属性、生命骰、技能熟练

#### Scenario: 生成背景选项
- **WHEN** AI随机生成开启且玩家进入背景选择
- **THEN** AI根据模板世界观、种族、职业生成3个背景选项
- **AND** 每个背景包含：名称、描述、技能熟练、特性

#### Scenario: 仅限人类模板的AI生成
- **WHEN** 模板设置为"仅限人类"（如现代都市恋爱模板）
- **THEN** 模板预设只有人类种族
- **AND** AI仍可生成3个任意种族选项（如：机器人、吸血鬼、外星人、克苏鲁等）
- **AND** 这些"非预期"的种族选项会产生有趣的剧情反差
- **AND** 增加游戏的惊喜感和可重玩性

### Requirement: 角色属性自动分配

系统应支持AI根据种族、职业、背景自动分配角色属性。

#### Scenario: 属性计算
- **WHEN** 玩家完成种族、职业、背景选择
- **THEN** AI计算最终属性值
- **AND** 属性值 = 基础值 + 种族加成 + 职业加成 + 背景加成
- **AND** 显示属性计算明细

### Requirement: 角色外观生成

系统应生成角色外观描述和文生图提示词。

#### Scenario: 生成外观描述
- **WHEN** 玩家完成角色创建
- **THEN** AI根据种族、职业、背景生成外观文字描述
- **AND** 同时生成文生图提示词（可在设置中关闭）

#### Scenario: 文生图提示词开关
- **WHEN** 设置中关闭文生图提示词
- **THEN** 角色卡不显示文生图提示词
- **AND** 后台仍生成提示词供未来功能使用

### Requirement: 角色创建导航

系统应支持玩家在角色创建过程中返回修改。

#### Scenario: 返回上一步
- **WHEN** 玩家在任意步骤点击"返回"
- **THEN** 返回上一步骤
- **AND** 保留之前的选择

#### Scenario: 从确认页返回
- **WHEN** 玩家在角色确认页点击"重新生成"
- **THEN** 返回种族选择步骤
- **AND** 清除后续选择

### Requirement: 设置项扩展

系统应新增角色创建相关设置项。

#### Scenario: AI随机生成开关
- **GIVEN** 玩家进入设置界面
- **WHEN** 切换"AI随机生成角色选项"开关
- **THEN** 保存设置
- **AND** 下次创建角色时应用新设置

#### Scenario: 文生图提示词开关
- **GIVEN** 玩家进入设置界面
- **WHEN** 切换"生成文生图提示词"开关
- **THEN** 保存设置
- **AND** 角色卡显示/隐藏提示词

## MODIFIED Requirements
