# Checklist

## 主菜单入口
- [x] 主菜单显示「模板管理」按钮，位于「设置」按钮上方
- [x] 点击「模板管理」按钮可进入模板管理界面

## 模板管理界面
- [x] 显示所有模板卡片，包含名称、描述、游戏模式、标签、作者
- [x] 内置模板显示「内置」标记
- [x] 自定义模板显示「自定义」标记
- [x] 「新建模板」按钮可创建空白模板
- [x] 自定义模板显示「编辑」按钮
- [x] 内置模板显示「查看」按钮（只读模式）
- [x] 所有模板显示「复制」按钮

## 模板编辑器 - 基础功能
- [x] 左侧导航可切换编辑模块

- [x] 取消按钮可退出编辑（有未保存更改时提示）
- [x] 内置模板为只读模式，不显示保存按钮

- [x] 可编辑模板名称、描述、版本、作者

- [x] 可编辑世界名称、描述、时代背景

- [x] 可编辑科技水平
- [x] 可添加/删除自定义字段（键值对）

## 种族编辑器
- [x] 显示种族列表

- [x] 可删除种族

## 职业编辑器
- [x] 显示职业列表

- [x] 可删除职业

## 背景编辑器
- [x] 显示背景列表
