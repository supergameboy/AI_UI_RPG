## 测试步骤

### 第一阶段：基础功能测试

#### 1.1 主菜单测试

1. 启动应用，验证主菜单显示
2. 测试"开始新游戏"按钮 → 应跳转到模板选择
3. 测试"读取存档"按钮 → 应打开存档管理器
4. 测试"设置"按钮 → 应打开设置面板
5. 测试"开发者模式"开关 → 应显示/隐藏开发者工具

#### 1.2 设置系统测试

1. 打开设置面板
2. 配置 LLM 提供商（DeepSeek/GLM/Kimi/OpenAI）
3. 输入 API Key
4. 保存设置
5. 刷新页面，验证设置是否持久化

#### 1.3 开发者工具测试

1. 开启开发者模式
2. 验证开发者面板显示
3. 检查 WebSocket 连接状态
4. 查看各 Tab 是否正常工作

### 第二阶段：模板系统测试

#### 2.1 模板选择测试

1. 点击"开始新游戏"
2. 验证模板选择界面显示
3. 检查 4 个预设模板是否显示
4. 选择一个模板，验证跳转到角色创建

#### 2.2 模板编辑器测试

1. 创建新模板
2. 编辑基本信息
3. 编辑世界观
4. 编辑种族/职业/背景定义
5. 保存模板
6. 预览测试

### 第三阶段：角色创建测试

#### 3.1 名称输入测试

1. 进入角色创建界面
2. 输入角色名称
3. 验证进入种族选择

#### 3.2 AI 生成测试（需配置 LLM）

1. 开启"AI 随机生成"选项
2. 验证种族选项生成（3 个选项）
3. 选择种族，验证职业选项生成
4. 选择职业，验证背景选项生成
5. 选择背景，验证属性计算

#### 3.3 属性计算测试

1. 验证基础属性值
2. 验证种族加成/惩罚
3. 验证职业加成
4. 验证最终属性值

#### 3.4 角色确认测试

1. 查看完整角色卡
2. 确认角色
3. 验证进入游戏界面

### 第四阶段：存档系统测试

#### 4.1 存档创建测试

1. 在游戏中创建存档
2. 输入存档名称
3. 验证存档保存成功

#### 4.2 存档加载测试

1. 返回主菜单
2. 点击"读取存档"
3. 选择存档
4. 验证加载成功

#### 4.3 存档导入/导出测试

1. 导出存档为 JSON
2. 删除存档
3. 导入 JSON 文件
4. 验证存档恢复

### 第五阶段：开发者工具深度测试

#### 5.1 请求监控测试

1. 执行 AI 生成操作
2. 查看请求监控 Tab
3. 验证请求/响应记录

#### 5.2 日志系统测试

1. 查看日志 Tab
2. 验证后端日志实时推送
3. 检查日志文件生成

#### 5.3 状态检查测试

1. 查看状态检查 Tab
2. 验证游戏状态显示
3. 尝试修改状态
